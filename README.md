# Utility Scripts

A collection of Python utility scripts for everyday automation tasks.

## Scripts
| Script | Description |
|--------|-------------|
| `file_organizer.py` | Organize files in a directory by extension |
| `csv_analyzer.py` | Quick CSV stats and column summaries |
| `password_gen.py` | Secure random password generator |
| `json_formatter.py` | Pretty-print and validate JSON files |

## Requirements
```
Python 3.8+
```

## Usage
```bash
python file_organizer.py /path/to/messy/folder
python csv_analyzer.py data.csv
python password_gen.py --length 16 --special
```
